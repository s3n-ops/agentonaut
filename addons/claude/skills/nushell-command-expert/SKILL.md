# Nushell Command Expert

## Instructions

- Use reconnect if `mcp__mcp-nushell` is not connected.
- Use `mcp__mcp-nushell__list_commands` to list all available Nushell native commands.
- Use `mcp__mcp-nushell__list_commands` with the `find` parameter to find a specific Nushell native command by searching command names, descriptions and search terms.
- Use `mcp__mcp-nushell__command_help` with the `name` parameter to get help for a specific Nushell command.

## Examples

- What Nushell command filters tables? → Search with "filter" or "table"
- How do I use the `each` command? → Get help for specific command
- List all filesystem commands → Search with "filesystem" or "directory"
